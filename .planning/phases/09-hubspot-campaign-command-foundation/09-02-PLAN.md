# Plan 09-02: HubSpot Campaign Persistence Model

## Objective
Introduce stable campaign-shell persistence and lifecycle update helpers.

## Tasks
1. Add `hubspot_campaigns` schema and CRUD/update commands in DB manager.
2. Add runtime passthrough for campaign create/list/get/update flows.
3. Add idempotency and lifecycle regression checks.

## Exit Criteria
- Campaign create is idempotent by name.
- Lifecycle state and `hubspot_campaign_id` persist/read correctly.
