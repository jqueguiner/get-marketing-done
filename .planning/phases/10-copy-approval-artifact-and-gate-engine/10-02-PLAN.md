# Plan 10-02: Approval Invalidation and Gate Enforcement

## Objective
Invalidate stale approvals on copy edits and block launch-adjacent actions.

## Tasks
1. Invalidate approval artifact when campaign email copy changes.
2. Enforce preflight/launch block when approval is missing or stale.
3. Validate invalidation path in deterministic regression script.

## Exit Criteria
- Editing approved copy causes approval status invalidation.
- Preflight and launch return blocked code until re-approved.
