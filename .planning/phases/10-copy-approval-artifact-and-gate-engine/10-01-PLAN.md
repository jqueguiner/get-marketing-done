# Plan 10-01: Copy Approval Artifact Schema and Commands

## Objective
Persist explicit human approval for campaign copy and expose operator controls.

## Tasks
1. Add approval artifact storage keyed by campaign.
2. Add `approve` and `approval-status` command modes.
3. Bind approvals to deterministic copy hash.

## Exit Criteria
- Approval records include reviewer and timestamp metadata.
- Approval status returns machine-readable validity state.
